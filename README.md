# Cursor + Gemini + Codex Setup

## Tools Installed
- Cursor IDE — AI-powered code editor (cursor.com)
- Gemini Code Assist — Google's AI coding extension for Cursor
- Codex by OpenAI — AI code completion and agent extension for Cursor

## Steps Completed
1. Downloaded and installed Cursor IDE from cursor.com
2. Installed Gemini Code Assist extension and signed in with Google account
3. Installed Codex by OpenAI extension and signed in with OpenAI account
4. Created a public GitHub repository named Cursor-gemini-codex
5. Installed Git on Windows (version 2.54.0)
6. Cloned the repository into Cursor using Git via terminal
7. Opened the repository folder in Cursor IDE

## Issues Encountered and Solutions
- Cursor's initial interface did not show an Extensions panel — 
  resolved by accessing extensions through the top menu bar
- Git was not pre-installed on Windows — 
  downloaded and installed from git-scm.com, then verified via terminal
- Codex extension did not auto-redirect for login — 
  resolved by opening Codex Sidebar manually through the Command Palette

## Notes
- Gemini Code Assist requires a Google account for authentication
- Codex requires an OpenAI account for authentication
- Git must be installed separately on Windows before cloning repositories
